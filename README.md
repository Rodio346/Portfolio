# Vishal Yadav Portfolio

Live site: [https://rodio346.github.io/Porfolio/](https://rodio346.github.io/Porfolio/)

## Run locally

- `npm install`
- `npm run dev`
- open `http://localhost:4321/Porfolio/`
- professional route: `http://localhost:4321/Porfolio/professional/`
- personal route: `http://localhost:4321/Porfolio/about-me/`

## Build

- `npm run build`
